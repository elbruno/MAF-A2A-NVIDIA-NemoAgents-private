# Blog Hero Prompts (16:9) — Detailed

Use target output close to `1600x900` (or higher 16:9, then downscale).

## Prompt A (recommended, cinematic + editorial)

```text
Create a high-end 16:9 hero image for a technical engineering blog post about a multi-agent architecture.

SCENE GOAL:
- Show a left-to-right workflow with three primary zones:
  1) Left: "NeMo Analysis Agent" represented by analytics screens, time-series charts, anomaly markers, KPI cards.
  2) Center: "Web Chat Orchestrator" represented by a modern chat panel with subtle message bubbles and routing arrows.
  3) Right: "MAF Action Agent" represented by action cards (alerts, report generation, execution status).
- Add thin telemetry/trace lines in the background to imply OpenTelemetry and Aspire observability.
- Convey motion: Analyze -> Route -> Act.

ART DIRECTION:
- Professional, editorial, cloud-native enterprise visual language.
- Lighting: soft dramatic rim light, clean contrast, no harsh bloom.
- Palette: deep navy, cobalt, cyan accents, tiny amber highlights for alerts.
- Composition must leave clean negative space in the top-left for title text overlay.
- Keep details sharp but uncluttered; prioritize readability at thumbnail size.

STYLE CONSTRAINTS:
- No logos, no watermarks, no stock-photo look, no cartoon style.
- No random text gibberish; UI-like shapes are okay but avoid fake readable text blocks.
- Avoid human faces as focal point; architecture/workflow should be the hero.
```

## Prompt B (clean diagrammatic, product-marketing style)

```text
Generate a polished 16:9 product hero visual for a GitHub repository showcasing cross-platform agent communication.

VISUAL STRUCTURE:
- Use a layered split composition:
  - Left panel: data analysis visuals (graphs, trends, anomaly spikes) for NeMo.
  - Middle panel: orchestration/chat command center.
  - Right panel: operational actions (alerts, reports, task completion states) for MAF.
- Add directional connectors with subtle glow to indicate A2A communication links.
- Add a faint infrastructure backdrop hinting at Aspire dashboard/resource health.

LOOK & FEEL:
- Modern B2B software campaign style.
- Crisp UI-inspired geometry, subtle depth, light volumetric haze.
- Color system: dark background + blue/cyan primary + violet secondary.
- High-fidelity, print-quality rendering.

NEGATIVE PROMPT / AVOID:
- Avoid fantasy/sci-fi over-stylization.
- Avoid clipart icons and generic blockchain visuals.
- Avoid excessive text, logos, brand marks, and watermark artifacts.
```
