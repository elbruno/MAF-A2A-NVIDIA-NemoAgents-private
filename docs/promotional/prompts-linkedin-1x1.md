# LinkedIn Image Prompts (1:1) — Detailed

Use `1200x1200` (or higher square) for better social quality.

## Prompt A (recommended, feed-friendly)

```text
Design a premium square LinkedIn visual (1:1) for a software engineering post about multi-agent systems.

CORE STORY:
- Present three components in a clear triangular or horizontal relationship:
  - NeMo Analysis Agent (data analytics visual language)
  - Web Chat Orchestrator (chat + routing control)
  - MAF Action Agent (alerts, report execution, action completion)
- Show directional flow arrows with subtle glow: Analyze -> Route -> Act.
- Include discreet observability motifs (trace lines, health indicators) in the background.

COMPOSITION:
- Keep central focal point strong for mobile feed preview.
- Reserve ~20% clean area for optional text overlay.
- Ensure object spacing and hierarchy are clear at small scale.

STYLE:
- Enterprise SaaS campaign style, modern and confident.
- Dark theme with blue/cyan/violet accents.
- Crisp details, moderate depth, no visual noise.

RESTRICTIONS:
- No brand logos, no watermark, no fake readable paragraphs.
- Avoid cartoon characters and over-abstract blobs.
```

## Prompt B (visual metaphor style)

```text
Create a square (1:1) technology key visual showing two specialized AI agents collaborating through a central orchestration hub.

METAPHOR:
- Left side = intelligence/analysis (charts, anomaly pulses, KPI metrics).
- Right side = execution/actions (alerts, reports, task pipelines).
- Center = conversation/orchestration interface that routes intent.
- Connect all components with clean A2A protocol-style links.

ART DIRECTION:
- Premium digital illustration, realistic UI-inspired elements.
- Balanced contrast and modern gradient lighting.
- Designed specifically for LinkedIn engagement and clarity.

AVOID:
- Meme style, stock-photo people, noisy typography, random text artifacts.
```
